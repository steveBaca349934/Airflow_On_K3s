# Airflow_On_K3s
## Offboarding cronjobs utilized by TradePro Reimagined to a local airflow environment

### These cronjobs scrape internet off the web and write data about 4 asset classes to a Postgress DB
* Equities Data
* Mutual Fund
* ETF Data
* Crypto Data
